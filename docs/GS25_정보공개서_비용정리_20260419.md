## ⑦ 가맹수수료 (로열티)

| 구분 | 내용 |
|---|---|
| 산정 기준 | 매출총이익 = 매출액 − 매출원가 |
| 수수료율 | ⚠️ **영업비밀 사유로 정보공개서 원본 비공개** (열람 요청 필요) |

| 제휴서비스 수수료 | 가맹수수료 배분율대로 분담 |

| 관점 | 시사점 | 후속 액션 |

| **월매출 기준값** | 서울 GS25 평균 월 6,260만원 vs LOKI 서울 편의점 평균 월 3,190만원 → 약 2배 차이 | ×1.94 보정 논리 재검토 필요 |
