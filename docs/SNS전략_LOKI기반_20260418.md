# LOKI 데이터 기반 SNS 전략
**작성일:** 2026-04-18  
**운영자:** 김명수 (AI시대 1인 혁신가)  
**목표:** 전 세계 천만 팔로워 → 2차활동(로컬온 플랫폼) 인지도 기반 구축

---

## 핵심 포지셔닝

> **"AI와 데이터로 창업의 진실을 말하는 사람"**

대부분의 창업 콘텐츠는 성공담, 동기부여, 감으로 가득하다.  
명수님은 반대다. **숫자로 말한다.**  
서울 25개구 56개 업종의 실제 수익 데이터를 가진 사람은 극소수다.  
그 희소성이 포지셔닝의 핵심이다.

**타겟 페르소나:**
- 창업을 고민하는 30~50대 직장인
- 이미 창업했지만 수익에 의문을 품는 소상공인
- 투자자·부동산 관련 종사자 (상권 데이터 필요)
- AI 비즈니스·데이터 활용에 관심 있는 사람

---

## 채널별 역할 분담

| 채널 | 역할 | 핵심 포맷 | 성장 속도 |
|------|------|-----------|-----------|
| **인스타그램** | 데이터 시각화 + 바이럴 | 카드뉴스, 인포그래픽 | 빠름 |
| **X (트위터)** | 실시간 인사이트 + 네트워킹 | 스레드, 숫자 트윗 | 중간 |
| **유튜브** | 심층 분석 + 신뢰도 구축 | 10~15분 분석 영상 | 느리지만 강력 |
| **블로그/브런치** | SEO + 아카이브 | 2,000~3,000자 리포트 | 검색 유입 |

## LOKI 데이터로 만들 수 있는 콘텐츠 유형

### 🔥 HIGH IMPACT (바이럴 가능)

**1. "서울에서 가장 수익성 높은 업종 TOP5"**
- 인스타 카드뉴스: 슬라이드 5장, 각 업종 순이익 공개
- X 스레드: "편의점 vs 카페, 뭐가 더 남을까? 데이터로 보니..."
- 유튜브: 15분 분석 영상 + 계산 공식 공개

**2. "강남 vs 노원, 같은 업종인데 수익이 2배 차이 나는 이유"**
- 지역별 LOOKUP 데이터 → 구별 비교 인포그래픽
- 임대료 지수(a값) + 유동인구 + 순이익 매트릭스

**3. "카페 창업, 월 순이익 진짜 얼마일까? (AI가 계산함)"**
- LOKI 실제 계산 결과 스크린샷 + 해설
- 댓글 참여 유도: "당신이 생각한 금액과 비교해보세요"

**4. "월 100만원도 못 버는 업종 리스트"**
- 충격적 숫자로 어텐션 확보
- 반전 메시지: "그래도 잘 버는 사람이 있는 이유"

**5. "창업 전 반드시 확인해야 할 3가지 숫자"**
- 월매출, 원가율, 손익분기점
- LOKI를 통해 직접 계산하는 방법 안내 → 유입 유도

### 📊 CONSISTENT (정기 콘텐츠)

**매주 월요일: "이번 주 창업 데이터 인사이트"**
- 특정 업종 1개 심층 분석
- 인스타 + 블로그 연동

**매주 수요일: "Q&A 데이터 답변"**
- 팔로워 질문 → LOKI 데이터로 답변
- X 스레드 형식

**매주 금요일: "이번 주 창업 뉴스 팩트체크"**
- 언론에 나온 창업 뉴스를 데이터로 검증
- "기사에서 말한 것 vs 실제 데이터"

## 콘텐츠 제작 워크플로우 (1~2시간/일)

```
[30분] 기획·데이터 추출
  → LOKI에서 해당 업종/구 데이터 추출
  → 인사이트 포인트 3개 선정
  → 제목 후보 3개 작성 (숫자 포함 필수)

[20분] 제작 (AI 활용)
  → 카드뉴스: Canva AI 또는 Claude에게 디자인 지시
  → X 스레드: Claude가 초안 작성 → 명수님 톤으로 수정
  → 블로그: Claude 초안 → 명수님 경험·관점 추가

[10분] 게시 + 크로스포스팅
  → 인스타 → X에 링크 공유 → 블로그에 롱폼 버전
  → 해시태그: #창업데이터 #소상공인 #로컬온 #AI창업분석

[매주 1회 30분] 유튜브 촬영
  → 그 주 가장 반응 좋은 인스타/X 글 → 영상으로 확장
  → 스크린 녹화 + 얼굴 노출 없이 시작 가능
```

## 주간 콘텐츠 스케줄

| 요일 | 인스타 | X | 유튜브 | 블로그 |
|------|--------|---|--------|--------|
| 월 | 카드뉴스 1편 | 스레드 1개 | - | 롱폼 리포트 |
| 화 | 스토리 (데이터 퀴즈) | 단문 2~3개 | - | - |
| 수 | - | Q&A 스레드 | - | - |
| 목 | 릴스 (15초 숫자 영상) | 리트윗+코멘트 | - | - |
| 금 | 카드뉴스 1편 | 팩트체크 스레드 | 업로드 | - |

**총 시간:** 평일 1~1.5시간 + 주말 0.5시간 = 약 8~10시간/주

## 바이럴 콘텐츠 공식

```

예시:
✅ "서울 25개구 분석했더니, 카페가 편의점보다 3배 덜 남았다"

[후킹 전략]
첫 줄에 숫자 → 중간에 반전 → 끝에 행동 유도(LOKI 링크)
```

## 단계별 성장 목표

### Phase 1: 씨앗기 (0~3개월)
- **목표:** 각 채널 팔로워 1,000명
- **전략:** 양보다 질, 10개 게시물 중 1개 바이럴 노리기
- **핵심 액션:** 창업 커뮤니티(네이버카페, 디시창업갤) 기존 질문에 LOKI 데이터로 댓글 달기
- **지표:** 저장수 > 좋아요 (인스타), 리트윗 > 좋아요 (X)

### Phase 2: 성장기 (3~6개월)
- **목표:** 인스타 5,000 / X 3,000 / 유튜브 1,000 구독
- **전략:** 협업 (창업 전문가, 부동산 전문가와 콜라보)
- **핵심 액션:** "창업 데이터 기반 인터뷰" 시리즈 시작
- **지표:** LOKI 월간 방문자 1,000명 → SNS 유입

### Phase 3: 도약기 (6~12개월)
- **목표:** 인스타 2만 / X 1만 / 유튜브 5,000
- **전략:** 유료 콘텐츠 또는 컨설팅 상품화 → 로컬온 플랫폼 연결
- **핵심 액션:** "창업 수익성 진단 리포트" PDF 무료 배포 (이메일 수집)
- **전환:** SNS 팔로워 → 로컬온 서비스 유저로

## AI 자동화 활용 방안

현재 완전 자동화(SNS 직접 게시)는 제약이 있지만, 제작 워크플로우 자동화는 가능합니다.

**Claude 활용 가능 작업:**
- 콘텐츠 초안 작성 (X 스레드, 블로그 초안)
- LOKI 데이터 해석 → 인사이트 문장화
- 해시태그 최적화 분석
- 경쟁 채널 벤치마킹 자료 조사

**반자동화 도구 추천:**
- Buffer / Later: 사전 예약 게시
- Canva: AI 이미지 생성 + 카드뉴스 템플릿
- Notion: 콘텐츠 캘린더 관리

- 주식 + 플랫폼 비즈니스를 동시에 도전하는 현실적 여정 공유

- "AI시대 새로운 사업가 모델"을 직접 증명하는 과정 자체가 콘텐츠
